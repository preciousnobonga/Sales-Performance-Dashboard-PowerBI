# Sales Performance Dashboard (Power BI)

## Business Context
Executives require real-time visibility into revenue trends, product performance, and regional sales to support strategic decision-making.

## Problem Statement
Develop an interactive dashboard to track key sales KPIs and identify performance trends across products and regions.

## Dataset Overview
- Sales transactions
- Product categories
- Regional data
- Revenue and quantity metrics
- Time dimension

## Tools Used
- Power BI
- DAX
- Excel (data preparation)

## Dashboard Features
- Revenue trend analysis
- KPI tracking (Total Sales, Growth %, Profit Margin)
- Product performance breakdown
- Regional performance comparison
- Interactive slicers for dynamic filtering

## Key Insights
- Seasonal revenue spikes in Q4.
- Certain product categories consistently outperform others.
- Regional sales disparities highlight growth opportunities.

## Business Recommendations
- Increase marketing investment in high-growth regions.
- Re-evaluate underperforming product lines.
- Plan inventory according to seasonal trends.
